#  Web Development Projects Collection

This repository contains three beginner-to-intermediate level web development projects built using **HTML, CSS, and JavaScript**.

---

#  Projects Included

## 1️⃣  Image Gallery

A responsive image gallery with lightbox view and category filters.

###  Features

* Responsive grid layout
* Lightbox preview with next/prev navigation
* Hover effects & smooth transitions
* Category-based filtering (e.g., Nature, City)

###  Technologies Used

* HTML
* CSS (Grid, animations)
* JavaScript (DOM manipulation)

---

## 2️⃣  Calculator

A functional calculator supporting basic arithmetic operations.

###  Features

* Addition, subtraction, multiplication, division
* Clear (C) and delete (DEL) functions
* Real-time input display
* Keyboard support

###  Technologies Used

* HTML
* CSS (Grid layout, UI styling)
* JavaScript (event handling, logic)

---

## 3️⃣  Portfolio Website

A personal portfolio website to showcase skills and projects.

###  Features

* Sections: About, Projects, Skills, Contact
* Smooth scrolling navigation
* Hover effects and animations
* Fully responsive design

### Technologies Used

* HTML
* CSS (Flexbox, responsive design)
* JavaScript (scroll animations)

---


# ▶️ How to Run

1. Download or clone the repository:

   ```
   git clone https://github.com/yourusername/projects.git
   ```
2. Open any project folder
3. Run the `index.html` file in your browser

---

#  Deployment

## GitHub Pages

* Go to repository settings
* Enable GitHub Pages from the main branch

## Netlify

* Drag and drop the project folder on Netlify
* Your site will be live instantly
